Modern QR Code Attendance Solution

This web application streamlines attendance for students using secure QR code technology. Students can sign up, download their unique QR, and mark attendance in real-time. Admins can manage requests and monitor attendance statistics with ease.

  • Fast, contactless attendance marking
  • Real-time statistics and reporting
  • Secure student authentication
  • Admin approval for new signups
  • Fully cloud-based with MongoDB Atlas
